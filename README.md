# Clinica dr Enias

Desafio laçado no meu curso de Análise e desenvolvimento de sistemas do SENAI para criar um site de uma clínica médica ao qual precisava ter 3 tipos de usuário (médico, paciente e adiministrador) sem utilizar Javascript. Utilizando apenas html e css. 

Explicação: Ao clicar para entrar com uma conta, você possue 3 opções:

- Entrar: Você entra com o perfil do adiministrador
- Entrar com o google: Você entra com o perfil do médico
- Entrar com Apple: Você entra com o perfil do paciente

Processo de criação:

1 -> De início, realizamos um protótipo na plataforma "figma", software para designer de site = https://www.figma.com/proto/4oBywpojKQRr1QvSfFMZEj/Untitled?node-id=0-1&t=VwE7ez7q9HR5M8GE-1 .

2 -> Estudando um pouco mais sobre "flexbox", criamos os elementos pricipais como header, classes principais de cada página e recolhemos as imagens do protótipo.

3 -> Como não poderiamos utilizar javascript, estudei como poderia trocar de página por meio da tag "<a></a>" assim, por mais que as páginas nao estivessem completas, já era possível trocar de páginas.

4 -> Reajustamos a paleta de cores e recolhemos a fonte original (Alata) do protótipo, adicionamos alguns detalhes como hover animado e reajuste de algumas imagens.
